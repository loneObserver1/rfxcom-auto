# RFXCOM pour Home Assistant

Intégration complète pour contrôler vos appareils RFXCOM via les protocoles AC et ARC.

## Caractéristiques principales

- ✅ Support des protocoles AC et ARC
- ✅ Gestion des interrupteurs ON/OFF
- ✅ Mode appairage intégré
- ✅ Interface de configuration intuitive
- ✅ Compatible HACS

## Installation rapide

1. Installez via HACS
2. Redémarrez Home Assistant
3. Ajoutez l'intégration dans Configuration > Intégrations
4. Configurez votre port série
5. Appairez vos appareils
